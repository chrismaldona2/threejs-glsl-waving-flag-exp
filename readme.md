# Three.js Waving Flag with Custom Shaders (GLSL)

A simple waving flag animation built using Three.js and custom GLSL shaders.

### Controls

- **Press "f" or double-click** to toggle fullscreen.
- **Press "h"** to hide/show GUI tweaks.
